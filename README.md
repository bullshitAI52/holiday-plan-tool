# Excel 高级比对工具 v1.3

一个功能强大的 Excel 文件比对工具，支持多工作表切换、修改和下载。

## 功能特点

- 📊 支持 Excel 文件（.xlsx, .xls）和 CSV 文件
- 🔄 双文件并排比对，差异高亮显示
- 📑 多工作表切换支持
- ✏️ 在线编辑和修改
- 💾 下载比对结果
- 📈 实时统计比对摘要
- 🔀 支持逐行对比和跨行对比两种模式
- 📜 历史记录功能

## 本地使用

1. 克隆或下载本仓库
2. 直接在浏览器中打开 `index.html` 文件
3. 上传两个 Excel 文件开始比对

## 使用说明

1. **上传文件**：点击左右两侧的上传区域，分别上传原始文件和对比文件
2. **开始比对**：点击"开始比对"按钮
3. **查看结果**：差异单元格会以不同颜色高亮显示
4. **切换工作表**：如果 Excel 文件包含多个工作表，可以通过下拉菜单切换
5. **编辑内容**：双击单元格可以直接编辑
6. **保存修改**：点击"保存修改"按钮保存编辑内容
7. **下载结果**：点击"下载表格"按钮下载比对结果

## 技术栈

- HTML5
- CSS3
- JavaScript (原生)
- SheetJS (xlsx.js) - Excel 文件处理
- JSZip - 文件压缩
- FileSaver.js - 文件下载
- Font Awesome - 图标库
